Automatización con n8n + IA + Cloudflare Pages

Generá propuestas web a partir de un Instagram

Esta página está lista para publicarse en Cloudflare Pages. La idea es que n8n cree nuevas carpetas por cliente y las publique automáticamente desde GitHub.

Ver demo de cliente Ver proceso

Landing personalizada

Colores, estilo, servicios, propuesta y botón directo a WhatsApp.

Cómo funcionaría la automatización

1

Recibís el Instagram

El cliente o vos cargan el link del perfil, nombre del negocio y rubro.

2

La IA genera la propuesta

Gemini o ChatGPT arma texto, estructura, colores y una landing simple.

3

Se publica solo

n8n sube los archivos al repo y Cloudflare Pages crea el link público.

Demo incluida

La carpeta trae una demo lista en /clientes/demo/. Después podemos hacer que n8n cree carpetas como /clientes/barberia-el-corte/ o /clientes/panaderia-dulce/.
